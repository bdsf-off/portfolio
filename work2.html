Landing Page WordPress

    Ce site est une landing page d'une application mobile qui présente le projet d'App Design Grow’up
    Ce projet a été réalisé avec l'outil de gestion de contenu WordPress. Le thème qui a été choisi pour ce projet est OceanWp et a été construit grâce à Elementor.
    N.B. : le projet étant hébergé sur un serveur de développement gratuit, vous pouvez voir s'afficher une erreur 504. En cas d'erreur, rafraichissez la page avec les touches CTRL Shift R.

  • Outils WordPress Boostrap PHP
  • Typographies "Montserrat Alternates" et "Montserrat"
  • Choix Couleurs

Construction de la page avec un page builder

Le page builder utilisé est Elementor le constructeur de pages pour WordPress La landing page devait reprendre approximativement le même design que le projet de design d'application Grow'Up.

Custom Post Type

Le custom post type permet de créer du contenu différent grâce à des champs personnalisés qui se differencient par rapport aux champs fournis par WordPress.
Ce dernier est créer direcctement dans le fichier function.php du thème WordPress en y rajoutant une fonction contenant les champs souhaités.
Les champs créés dans Pour cette landing page sont des champs qui constituent une fiche de recette de cuisine avec toutes les informations nécéssaires pour la recette.
